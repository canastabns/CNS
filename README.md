|Dependency|Version|
|:---------|:-----:|
|Node      | 10.24.1 (REQUIRED)|
|NPM       | 6.14.12 |

# Getting started for install

Install Truffle
``$ npm install -g truffle``

To run unit tests, clone this repository, and run:
``
$ npm install
$ npm test
$ npm run deploy
``

Verify files
``truffle run verify ENSRegistry@$CONTRACT_ADDRESS --network testnet``
